# Plugins Index

!!! abstract "In a nutshell"
    OpenVoiceOS is built from interchangeable building blocks called *plugins* — small add-ons that each handle one job, like turning speech into text or text into speech. This works much like apps on a phone: you can mix and match the pieces you want and swap them out later. This page lists every kind of plugin available. See the [Glossary](glossary.md) for related terms.

OpenVoiceOS is built on a highly modular plugin architecture. This index provides a complete list of all plugins available in the workspace.
